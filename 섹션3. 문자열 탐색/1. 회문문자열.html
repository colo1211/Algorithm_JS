방법 1

대소문자 구분 하지 않는 것은 그냥 대문자로 통일하자! : toUpperCase()

반복문을 돌 때, 마지막 종료조건은 문자열 길이의 절반 => Math.floor(n/2)

Math.floor 하지 않으면 소수점이되는데 돌아가긴 한다.

길이가 홀수인 문자열도 중간의 값은 상관없으므로 처리하는 방법은 길이가 짝수인 것과 동일하다.

방법 2 : reverse()

reverse() -> 배열을 반대로 뒤집는 메소드 (반드시 배열에서만 적용 가능)

s.split('').reverse().join('')

  • 문자열 s를 배열로 만든다. : split
  • 배열로 만든 대상을 reverse 한다.
  • reverse된 배열을 합쳐서 문자열로 만든다.

문자열 -> 배열 : split

배열 -> 문자열 : join
